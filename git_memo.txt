git init
 .gitフォルダの作成　隠しフォルダだから注意


git add "ファイル名"
　特定のファイルの変更を次のコミットに含める　ステージにあげる

git commit 
　addされた変更をまとめて記録

git commit -m "コメント"

git status
　ステージ上にあるものをチェック


git restore --staged "ファイル名"
　ステージから外す　commit何もしてないときは使えない?

git rm "ファイル名"
　gitの管理下からファイルを削除して次のコミットでその削除を反映させる

git remote add origin https://github.com/nezumitori990/push_test
git push origin main
　リポジトリの更新
